# Archive

This folder stores legacy projects and historical scripts that are not part of
the active top-level CMAP module pipeline.

The directory `legacy_modules/` holds full copies of old segmentation projects
on this filesystem. It is listed in the repo root `.gitignore` because those
trees include large outputs and are not meant for normal git history.

Active modules remain directly under `cmap/`:
- `segmentation/`
- `postprocess/`
- `features/`
- `qc/`
- `visualization/`
- `napari-plugin/`
- `pipelines/`
